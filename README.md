# fastapi-prometheus-grafana
fastapi + prometheus + grafana

## Usage

Before starting the application, install Docker on your system if not already installed.

```bash
# start
docker compose -f ./docker-compose.yml up -d

# restart
docker compose -f ./docker-compose.yml restart
```
